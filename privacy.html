AI Prompt Builder

Privacy Policy

Last updated: March 6, 2026

1. Information We Do Not Collect

We believe in absolute privacy for your ideas. Therefore:

  • No Input Saving: We do NOT save, store, or log the text you enter into the AI Prompt Builder. It is processed in real-time and immediately discarded.
  • No Personal Information: We do not intentionally collect or store personal information such as names, email addresses, or the text you enter into the tool.

    This application only processes your input temporarily in order to generate a prompt and does not store that input in any database.

    Basic server or hosting logs may be automatically processed by infrastructure providers for operational and security purposes. These logs are not used by this application to identify users.
  • No Accounts: This tool operates entirely anonymously. There is no user account or login system.

2. Information We Do Collect (Anonymous Statistics)

To understand how our tool is used and to improve it, we collect completely anonymous usage statistics. This data cannot be traced back to you:

  • Event Types: We log when an action occurs (e.g., "generate", "copy", "share").
  • Broad Categories: We classify usage into broad, non-specific categories (e.g., "writing", "business", "coding").
  • Input Length Buckets: We log the rough size of the request (e.g., "small", "medium", "large") to optimize performance.
  • Timestamps: We record the time an event occurred.

The tool collects limited anonymous usage statistics such as:

  • generation events
  • copy button usage
  • share button usage
  • approximate input length buckets

These statistics are used only to understand general usage patterns and improve the service. No personal data or raw user input is stored.

3. Third-Party Services

We may use basic hosting and analytics providers to process the anonymous data mentioned above. These providers are bound by strict data processing agreements.

4. Changes to this Policy

We may update our Privacy Policy occasionally. We will notify you of any changes by posting the new Privacy Policy on this page.
